# Images Folder

## Profile Image

Please add your professional headshot image here with one of these filenames:
- `profile.jpg`
- `profile.png`
- `dhruval-patel.jpg`
- `dhruval-patel.png`

The image will be automatically displayed in the hero section of the portfolio website.

**Recommended specifications:**
- Format: JPG or PNG
- Size: At least 600x600 pixels (square format works best)
- File size: Under 2MB for optimal loading

After adding your image, make sure the filename in `index.html` (line 62) matches your image filename.
